# Análisis

## Descripción
Directorio que contiene todos los archivos generados durante el desarrollo del estudio.

## Subdirectorios

### 1. Ensamblajes
Contiene ensamblajes provisionales de secuencias genómicas generados con MIRA4.

### 2. QC (Control de Calidad)
- Informes de calidad generados con PRINSEQ
- Gráficos preliminares de calidad
- Reports de filtrado de secuencias

### 3. Taxonomia
Resultados taxonómicos provisionales:
- Análisis de gen 16S rRNA
- Clasificación de viroma
- Asignación taxonómica contra RDP database
