## Objetivos de la Reunión
La reunión tuvo como base revisar y definir los objetivos generales y específicos del proyecto IPC-2025-GRUPO5, así como discutir el plan de trabajo, las metodologías y roles dentro del equipo.

## Discusión del Proyecto IPC-2025-GRUPO5
Se planteó la importancia de estudiar el microbioma y viroma intestinal mediante técnicas avanzadas como secuenciación masiva y citometría de flujo. La atención se centró en optimizar protocolos para preparar librerías de secuenciación a partir de células seleccionadas por citometría, y el estudio detallado de partículas virales.

Se repasaron las técnicas clave que se utilizarán:
- Fijación con formaldehído y tinción seleccionada para bacterias y virus.  
- Extracción de ADN mediante método fenol-cloroformo con tratamiento enzimático (CTAB, lisozima, proteinasa K).  
- Amplificación dirigida con polimerasa Φ29 y amplificación de regiones V3-V4 del gen 16S rRNA con barcoding.  
- Fragmentación del ADN por sonicación o tagmentación (Nextera).  
- Secuenciación con plataformas Illumina y 454 FLX+.  
- Análisis bioinformático usando herramientas como PRINSEQ, MIRA4, SSAHA2, Bowtie2, BLAST y anotación con InterPro.  
- Visualización y análisis estadístico con R (vegan, ggplot2, FlowViz).

## Decisiones Tomadas

1. **Objetivo 1 - Protocolos de secuenciación:**  
   Optimizar la preparación de librerías para asegurar calidad y rendimiento, validando mediante comparación entre amplificación directa post-citometría vs. amplificación con Φ29.

2. **Objetivo 2 - Virómica dirigida:**  
   Definir procedimiento para aislamiento, tinción y secuenciación de partículas virales, con énfasis en control de contaminación bacteriana y recuperación eficiente de partículas.

3. Se acordó la estructura del repositorio para organizar eficazmente datos, scripts, análisis, resultados, y documentación.

4. Se definieron las tareas iniciales para la estandarización de protocolos de tinción y fijación, y para la implementación preliminar del pipeline de análisis bioinformático.

## Tareas y Próximos Pasos

- Documentar procedimientos estándar (SOPs) para extracción y amplificación.  
- Comenzar análisis preliminar con datos piloto y evaluar calidad con PRINSEQ.  
- Asignar responsables para cada área y calendarizar reuniones periódicas.

## Cierre
