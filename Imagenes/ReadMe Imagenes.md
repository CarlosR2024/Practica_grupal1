#  Imágenes

##  Descripción
Carpeta para todas las representaciones visuales del estudio.

##  Subdirectorios

### 1.  Graficas
Representaciones visuales generadas durante el análisis:
- Histogramas de calidad
- Coverage plots
- Plots de gating en citometría de flujo
- Gráficos de abundancia taxonómica

Ver resumen completo en: [`Graficas/RESUMEN_GRAFICOS.md`](Graficas/RESUMEN_GRAFICOS.md)

### 2.  Tablas
Representaciones gráficas de datos tabulares:
- Matrices de correlación
- Tablas de abundancia relativa
- Resúmenes estadísticos visualizados
