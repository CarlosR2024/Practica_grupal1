#  Metadata

##  Descripción
Información crítica de muestras, parámetros experimentales y logs.

##  Subdirectorios

### 1.  Citometria
Metadatos de ensayos de citometría de flujo:
- `Citometria_conf_laseres.xlsx`: Configuración de láseres
- `Fluorochrome_list.csv`: Lista de fluorocromos utilizados

### 2.  Muestras
Información crítica de las muestras:
- `Muestras.xlsx`: Tabla con ID | PARTICIPANTE | DÍA DE RECOGIDA

### 3.  Secuenciacion
Información detallada de cada corrida de secuenciación:
- `run_001_illumina.csv`: Metadata de corrida Illumina
- Parámetros de secuenciación
