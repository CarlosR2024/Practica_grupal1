Carpeta que contiene todos los programas del estudio. Se divide en 
1.**Citometria**: contiene codigos y programas para procesar, analizar datos de citometría
2. **R**: scripts en R para procesar y analizar
3. **Utiles**: paquetes utiles de trabajo
4. **Visualizaciones**: scripts dedicados a generar gráficos y visualizaciones de los datos
