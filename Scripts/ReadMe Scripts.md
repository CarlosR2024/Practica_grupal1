#  Scripts

##  Descripción
Carpeta que contiene todos los programas del estudio.

##  Subdirectorios

### 1.  Citometria
Códigos para procesar y analizar datos de citometría de flujo:
- Análisis con FlowViz
- Parámetros de gating
- Scripts de ordenamiento celular

### 2.  R
Scripts en R para procesamiento y análisis:
- `01_limpieza.R`: Limpieza de datos CSV
- `02_analisis.R`: Análisis estadístico
- `03_visualizacion.R`: Generación de gráficos

### 3.  Utiles
Paquetes y utilidades de trabajo:
- `requirements.R`: Dependencias del proyecto

### 4.  Visualizaciones
Scripts dedicados a generar gráficos:
- Plots de enterotipos
- Gráficos de calidad
- Visualizaciones de cobertura
