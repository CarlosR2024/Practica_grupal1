# Data - Gestión de Datos

## Descripción
Carpeta para guardar todos los datos usados en el estudio.

##  Estructura

### 1.  Raw (Datos Brutos)
Archivos CSV originales sin modificar:
- Archivo explicativo con datos incluidos
- Datos de secuenciación directa
- Exportaciones de citómetro
- Archivos FASTQ/FASTA originales

### 2.  Procesados (Datos Limpios)
Datos procesados y listos para análisis:
- Base de datos procesada que contiene edades, géneros y niveles relativos de 5 metabolitos para 10 muestras.
- Sin valores NaN
- Filtrados por calidad (PRINSEQ)
- Formateados para análisis downstream
